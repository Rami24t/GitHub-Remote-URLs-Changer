Git Multiple/New Remote-URL Processor

Put in your Repos names

Copy and paste your git-ssh-URLs in here:




To set future "push"es to your new Repo (only)


git remote add remote2Origin remote_2_url

git remote set-url --add --push remote2Origin remote_2_url

git push -u remote2Origin

git remote show

git status


To set the next "push"es to both repos at the same time


git remote add remote2Origin remote_2_url

git remote set-url --add --push remote2Origin remote_2_url

git remote add group remote_1_url

git remote set-url --add --push group remote_2_url

git remote set-url --add --push group remote_1_url

git push -u group

git status

git remote show

Adding the first remote origin:

git remote add origin remote_1_url

git remote set-url origin remote_1_url

Set the default remote branch for the current local branch

git branch --set-upstream main

or

git push -u origin main

Adding the second remote remote2Origin

git remote add remote2Origin remote_2_url

git remote set-url --add --push remote2Origin remote_2_url

Adding the common remote group

allows us to push to both at the same time

git remote add group remote_1_url

git remote set-url --add --push group remote_2_url

git remote set-url --add --push group remote_1_url

Pushing to remotes:

- To origin

git push origin main

- To repo-2

git push remote2Origin main

- To both

git push group main

Remove remote Url

git remote rm 'URL OF REMOTE'
